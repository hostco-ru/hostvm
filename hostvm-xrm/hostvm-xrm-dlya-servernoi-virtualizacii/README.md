# HOSTVM XRM (для серверной виртуализации)

#### X Recovery Manager <a href="#x-recovery-manager" id="x-recovery-manager"></a>

HOSTVM XRM предназначен для обеспечения непрерывной работы бизнес-критичных систем и автоматизации восстановления сервисов после сбоя.

HOSTVM XRM позволяет автоматизировать как простые сценарии так и сложные процессы восстановления для комплексной инфраструктуры виртуализации.
