# Системные требования

Стандартный виртуальный сервер архитектуры: **x86/x86-64.**

Количество ядер vCPU: **2**

Количество оперативной памяти: **2 ГБ**

Свободное место на жестком диске: **10 ГБ**



**Требования к инфраструктуре**

* Доступный брокер HOSTVM VDI
* Сетевая связность между HOSTVM XRM Director и управляемыми брокерами HOSTVM VDI
* Учетные данные с административными правами для доступа к брокерам HOSTVM VDI
* Основная и резервная площадка при использовании сценария миграции конфигурации
