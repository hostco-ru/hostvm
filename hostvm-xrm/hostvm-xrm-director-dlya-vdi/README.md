# HOSTVM XRM Director (для VDI)

HOSTVM XRM Director это инструмент для централизованного управления и автоматизации обеспечения катастрофоустойчивости вашей инфраструктуры HOSTVM VDI.

Позволяет обеспечить единый веб-интерфейс управления брокерами и автоматизированный перенос конфигураций между кластерами брокеров HOSTVM VDI, упростить и минимизировать ошибки при управлении и автоматизировать процессы миграции и подготовки резервных площадок.
