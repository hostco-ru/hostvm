# SAML

Global logout on exit — если включено, при выходе из системы на портале брокера, SAML так же отправляет запрос на выход IdP.

По умолчанию: выключено.

IDP Metadata Cache — продолжительность кэширования метаданных IDP (в секундах).

По умолчанию: 86400 (24 часа).

Organization display name — отображаемое наименование организации для использования в метаданных SAML SP.

Organization name — наименование организации для использования в метаданных SAML SP.

Organization URL — URL-адрес организации для использования в метаданных SAML SP.
