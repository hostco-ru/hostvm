# Custom

CSS — опциональный код для изменения стиля по умолчанию страниц web-портала брокера.

Logo name — текст, отображаемый рядом с изображением в верхнем левом углу стартовой страницы портала пользователя.

Min.Services to show filter — минимальное количество сервисов, отображаемых пользователю на странице портала, для появления опции поиска/фильтра.

Show Filter on Top — позволяет изменить расположение элемента поиска/фильтра сервисов на странице.

Site copyright info — текст, отображаемый в правом нижнем углу стартовой страницы портала пользователя.

Site copyright link — URL-адрес ссылки для текста параметра Site copyright info.

Site information — HTML-код для частичной кастомизации страницы входа в систему.

Введённый код будет отображаться под формой ввода учетных данных.

Site name — текст, отображаемый надо формой ввода учетных данных на странице входа в систему.
