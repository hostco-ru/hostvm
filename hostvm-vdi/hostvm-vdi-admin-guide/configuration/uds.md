# UDS

**UDS ID —** уникальный идентификатор инсталляции.

**autorunService —** при входе в систему сразу запускать назначенный пользователю сервис.&#x20;

Работает в случае, если пользователю назначен только один сервис. При активации скрывает экран выбора сервиса и использует первый по приоритету транспорт из доступных для сервиса.&#x20;

По умолчанию: выключено.&#x20;

**DisallowGlobalLogin —** если опция включена, при входе в систему не отображается глобальный список аутентификаторов.&#x20;

При включении данной опции, пользователи будут выполнять вход через аутентификатор "по умолчанию" или с наивысшим приоритетом. Чтобы выполнить вход через другие аутентификаторы, необходимо использовать "метку" в URL-адресе доступа (задается в аутентификаторе).

По умолчанию: выключено.

**KeepInfoTime —** определяет время, в течение которого завершённые события сервис-пулов остаются видимыми. Задается в секундах.

По умолчанию: 14401 (4 часа).

**RedirectToHttps —** при попытке доступа к брокеру по http автоматически перенаправлять на https.&#x20;

По умолчанию: выключено.

**SessionExpireTime —** максимальное время жизни сеанса пользователя после создания новой публикации.

По истечении данного времени система автоматически закроет пользовательскую сессию и приступит к удалению сервиса. Данный параметр не применяется, если в настройках используемого сервис-пулом менеджера ОС в качестве действия при выходе из системы выбрана опция **"Держать сервис привязанным даже в новой публикации"**.

По умолчанию: 24 часа.

**StatsDuration —** время, в течение которого система будет хранить статистику.

По умолчанию: 365 дней.
