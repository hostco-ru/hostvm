# Проброс USB-устройства с хоста в виртуальную машину

### Требования:

1. Убедитесь, что USB-устройство физически подключено к USB-порту сервера (хоста).
2. Определите, какая виртуальная машина (ВМ) должна получить доступ к этому устройству.
3. Важно: ВМ должна быть выключена. Если она работает, остановите её через интерфейс (кнопка Shutdown или Power Off).

### Пошаговая инструкция

1\. Перейдите к списку виртуальных машин, выбрав в меню Compute → Virtual Machines.

2\. Выберите требуемую виртуальную машину и нажмите на её имя. Откроется страница с деталями ВМ.

3\. Откройте вкладку «Host Devices». В списке будут отображаться уже проброшенные с хоста устройства для этой ВМ.

4\. Добавьте новое устройство нажав на кнопку Add Device (или «Добавить устройство»). Откроется диалоговое окно Add Host Devices.

5\. Настройте параметры проброса

* В поле Pinned Host убедитесь, что выбран правильный хост (обычно подставляется автоматически).
* В выпадающем списке Capability выберите значение usb\_device (для работы с USB-устройствами).
* В разделе Available Host Devices появится список всех доступных USB-устройств, подключённых к хосту.

6\. Выберите USB-устройство

* Найдите в списке нужное устройство.
* Примечание: если устройство уже используется другой ВМ, в колонках «Currently in use by» и «Attached to VMs» будет указано имя этой ВМ. Такое устройство нельзя добавить одновременно к нескольким ВМ.
* Отметьте устройство (щелчком по строке, флажком).
* Перенесите его в нижнюю таблицу Host Devices to be attached с помощью кнопки со стрелкой вниз: >>.

7\. Примените изменения нажав кнопку OK в диалоговом окне. Вы вернётесь на вкладку «Host Devices» — добавленное устройство должно появиться в таблице.

8\. Запустите виртуальную машину, нажав кнопку Run (или «Запустить») в верхней панели действий ВМ. Дождитесь полной загрузки гостевой ОС.

9\. Проверьте доступность устройства внутри ВМ. В гостевой ОС USB-устройство должно определиться автоматически. При необходимости установите драйверы (если устройство требует специфического ПО).
