# Активация домена хранения

Если при подключении домена хранения не был выбран дата-центр, домен получит статус «Unattached» (Не подключен). Этот статус означает отсутствие привязки к дата-центру, что делает домен недоступным для использования виртуальными машинами или хостами. Чтобы использовать такой домен, его необходимо активировать.

### Порядок действий

1. Перейдите в раздел управления хранилищами.\
   Откройте портал администрирования и выберите: Compute → Data Centers (или Storage → Data Centers).
2. Выберите целевой дата-центр.\
   Нажмите на дата-центр, к которому необходимо подключить домен (например, `Default`).
3. Откройте вкладку хранилищ.\
   В панели сведений выбранного дата-центра перейдите на вкладку "Хранилище" (или Storage).
4. Запустите процесс подключения нажав кнопку в соответствии с типом подключаемого домена:

* "Прикрепить данные" (или Attach Data).
* "Прикрепить ISO" (или Attach ISO).
* "Подключить экспорт" (или Attach Export).

5. Выберите домен.\
   В появившемся окне из списка доступных доменов выберите тот, который имеет статус "Unattached".
6. Завершите операцию.\
   Нажмите кнопку "ОК".

Выбранный домен хранения будет подключен к дата-центру и после автоматической активации станет доступен для использования.
