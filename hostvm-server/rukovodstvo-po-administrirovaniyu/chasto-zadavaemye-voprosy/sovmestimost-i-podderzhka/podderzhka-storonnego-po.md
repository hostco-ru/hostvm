# Поддержка стороннего ПО

В состав гипервизора HOSTVM включены пакеты для обеспечения совместимости со сторонними open-source решениями (такими как Ceph, Open vSwitch, GlusterFS). Данные компоненты поставляются для расширения функциональных возможностей интеграции, однако являются сторонним ПО и не покрываются технической поддержкой платформы HOSTVM.
