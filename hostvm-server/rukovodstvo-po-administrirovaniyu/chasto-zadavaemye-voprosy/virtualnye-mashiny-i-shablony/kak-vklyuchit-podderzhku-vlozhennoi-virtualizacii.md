# Как включить поддержку вложенной виртуализации?

Для включения вложенной виртуализации на виртуальной машины нужно выполнить следующие шаги:

1. Перейти в настройки виртуальной машины (Имя ВМ -> Edit) -> Вкладка Host -> Выбрать Specific Hosts(s), отметить галочками хосты, на которых может быть запущена машина.
2. Поставить галочку для опции Pass-Through Host CPU
