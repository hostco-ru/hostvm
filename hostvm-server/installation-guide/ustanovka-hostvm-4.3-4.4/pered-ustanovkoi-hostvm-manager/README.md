# Перед установкой HOSTVM Manager

Установка менеджера виртуализации производится на физический сетевой интерфейс. Порты коммутатора на время установки должны быть в режиме access. Конфигурирование VLAN’ов, объединение сетевых интерфейсов и т.д. производится после установки менеджера виртуализации в его веб-интерфейсе.
